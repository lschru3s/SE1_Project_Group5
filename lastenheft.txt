    Funktionale Anfordungen
    	Das System "Digitales Parkhaus" muss fähig sein die Zahl der parkenden Fahrzeuge zu erfassen.
    	Unter der Bedingung dass das Parkhaus nicht voll ist, muss das System "Digitales Parkhaus" fähig sein das Einfahren eines Fahrzeugs durch das Hochfahren der Schranke zu ermöglichen.
    	Unter der Bedingung dass das Parkhaus voll ist, muss das System "Digitales Parkhaus" fähig sein das Einfahren eines Fahrzeugs durch das Untenhalten der Schranken zu verwähren.
    	Das System Digitales Parkhaus muss fähig sein Parkgebühren entgegenzunehmen.
    	Unter der Bedingung dass kein Notfall vorliegt, muss das System "Digitales Parkhaus" fähig sein vor Ausfahrt eines Fahrzeugs die Bezahlprüfung durchzuführen.
    	Unter der Bedingung dass ein Notfall vorliegt oder die Bezahlprüfung positiv ausfiel, muss das System die Ausfahrt eines Fahrzeugs durch Hochfahren der Schranke ermöglichen.
		Das System "Digitales Parkhaus" soll die Möglichkeit bieten die einfahrenden Fahrzeuge auf freie Parkplätze hinzuweisen.

    	Das System "Digitales Parkhaus" soll die Möglichkeit bieten die Anzahl der freien Stellplätze abzurufen.
